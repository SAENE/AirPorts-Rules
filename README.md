# 代理规则
Clash/ 适用于[subconverter](https://github.com/tindy2013/subconverter)的订阅转换规则  
Clash/Clash-Auto-GeoSite.ini 完全依赖GeoSite文件和GEOIP文件，使用[Loyalsoldier/v2ray-rules-dat](https://github.com/Loyalsoldier/v2ray-rules-dat)或者[MetaCubeX/meta-rules-dat](https://github.com/MetaCubeX/meta-rules-dat)即可  
Clash/geosite 使用[meta-rules-dat](https://github.com/MetaCubeX/meta-rules-dat/tree/meta/geo/geosite)转换，删除Keyword项  
SingBox/ 适用于[sing-box-subscribe](ttps://github.com/NiuStar/sing-box-subscribe)订阅转换规则，SingBox-Route配置并未经过测试  
all_base.tpl 适用于[subconverter](https://github.com/tindy2013/subconverter)的订阅转换配置
